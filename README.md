## Project Overview

Arthsangh Finance Collective is a financial management application developed for Self-Help Groups (Bachat Gat) to manage member information, savings, transactions, and financial records efficiently. The system digitizes traditional record-keeping processes and provides a centralized platform for financial management.

## Technologies Used

- Spring Boot
- Java
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap (if used)

## Features

- Member Registration and Management
- Savings Record Management
- Deposit Tracking
- Transaction History
- Financial Reports
- Admin Dashboard
- Secure Login System
- Database Management

## System Modules

### Admin Module
- Manage members
- Monitor transactions
- Generate reports
- Manage savings records

### Member Module
- View savings details
- Track transaction history
- Update profile information

## Database

The application uses MySQL for storing:

- Member Details
- Savings Records
- Transactions
- User Accounts
- Financial Reports

## How to Run

1. Install Java JDK
2. Install MySQL Server
3. Create the database
4. Import the provided SQL file
5. Configure database settings in `application.properties`
6. Run the Spring Boot application
7. Open the application in your browser
